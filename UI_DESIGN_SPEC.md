## What is HOTMESS?

### The Vision

**HOTMESS is a Global Operating System for Gay Men.**

Not another dating app. Not another marketplace. Not another events platform. HOTMESS is the **singular, bold, luxury platform** that eliminates the need for multiple apps by bringing everything under one roof.

### The Platform

HOTMESS is a **multi-layered business ecosystem** serving:

| Layer | What It Does |
|-------|--------------|
| **Users** | AI-powered connections, multi-layered profile skins, compatibility matching |
| **Sellers** | P2P marketplace, verified seller program, escrow-protected transactions |
| **Buyers** | Shop official brands + creator products + ticket resale |
| **Creators** | Premium content, subscriptions, monetization tools |
| **Events** | Club nights, parties, venues - bridging the digital and physical club world |
| **Radio** | Live radio station with DJ shows, curated programming |
| **Record Label** | Music releases, artist features, audio drops |
| **Fashion** | In-house clothing brands, exclusive drops, merchandise |

### The Technology

- **AI & Data-Driven Connections** - 8-dimension compatibility scoring based on real data, not random swiping
- **The Globe (Pulse)** - Highly interactive 3D globe you can watch in real-time, showing people, events, and energy worldwide
- **Right Now** - Live availability status eliminating ghosting and uncertainty
- **Multi-Layered Profile Skins** - Profiles that adapt (Standard, Seller, Creator, Organizer, Premium)
- **AI Wingman** - Intelligent assistant for conversation starters, match insights, and profile optimization
- **Integrated Travel** - ETAs, directions, and Uber booking without leaving the app
- **Telegram Integration** - Feed chat connected directly to your Telegram
- **Safety System** - Check-ins, panic button, trusted contacts, aftercare - all built-in

## Brand Identity

### Brand Positioning
**HOTMESS is a Global Operating System** - not an app, not a platform, an *operating system* for gay men's digital lives. This positioning should inform every design decision.

### Brand Personality
- **Bold** - Unapologetic, statement-making, high contrast
- **Luxury** - Premium materials, editorial quality, fashion-forward
- **Global** - Operating system scale, not local app thinking
- **Inclusive** - For gay men, by gay men, celebrating the community
- **Multi-Dimensional** - Layers of functionality, not a single-purpose tool

### Brand Pillars

| Pillar | Expression |
|--------|------------|
| **BOLD** | Neon colors, oversized typography, statement visuals |
| **LUX** | Gold accents, premium textures, editorial photography |
| **GLOBAL** | The Globe as central metaphor, worldwide reach |
| **COMPLETE** | "One platform. Everything." - no compromises |

### Visual Language
- **Dark-mode first** - Black backgrounds as the canvas
- **Neon accents** - Hot pink, cyan, gold as signature colors
- **Glassmorphism** - Layered depth, blur, transparency
- **Editorial aesthetic** - Magazine-quality imagery and layouts
- **Bold typography** - Uppercase, wide tracking, statement headlines
- **Micro-animations** - Everything feels alive and responsive
- **The Globe** - Central visual metaphor tying everything together

## Animation & Motion

### Timing Functions
```css
/* Standard easing */
transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);

/* Bounce/Spring */
transition-timing-function: cubic-bezier(0.34, 1.56, 0.64, 1);

/* Smooth decelerate */
transition-timing-function: cubic-bezier(0, 0, 0.2, 1);
```

### Duration Scale
```
instant:  0ms (instant feedback)
fast:     150ms (micro-interactions)
normal:   200-300ms (standard transitions)
slow:     400-500ms (emphasis animations)
slower:   600-800ms (page transitions)
```

### Key Animations

#### Glow Pulse
```css
@keyframes glow-pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
/* Duration: 2s, Infinite */
```

#### Shimmer
```css
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
/* Duration: 2s, Linear, Infinite */
```

#### Float
```css
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}
/* Duration: 3s, Ease-in-out, Infinite */
```

#### Scale In
```css
@keyframes scale-in {
  from { transform: scale(0.9); opacity: 0; }
  to { transform: scale(1); opacity: 1; }
}
/* Duration: 300ms, Ease-out */
```

#### Slide Up
```css
@keyframes slide-up {
  from { transform: translateY(20px); opacity: 0; }
  to { transform: translateY(0); opacity: 1; }
}
/* Duration: 400ms, Ease-out */
```

#### Gradient Shift
```css
@keyframes gradient-shift {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}
/* Duration: 3s, Ease, Infinite */
/* Requires: background-size: 200% 200% */
```

#### Ripple
```css
@keyframes ripple {
  0% { transform: scale(1); opacity: 1; }
  100% { transform: scale(2); opacity: 0; }
}
/* Duration: 1s, Linear, Infinite */
```

### Framer Motion Presets

#### Spring Config
```javascript
spring: {
  type: "spring",
  stiffness: 300,
  damping: 30
}
```

#### Stagger Children
```javascript
container: {
  hidden: { opacity: 0 },
  show: {
    opacity: 1,
    transition: {
      staggerChildren: 0.1
    }
  }
}

item: {
  hidden: { opacity: 0, y: 20 },
  show: { opacity: 1, y: 0 }
}
```

### Globe Data Flows

#### Initial Load
1. Fetch beacons (500 limit, 30s refetch)
2. Fetch cities (60s refetch)
3. Fetch user intents (100 limit, 30s refetch)
4. Fetch Right Now users (15s refetch)
5. Get user location (geolocation API)
6. Fetch nearby people (15s refetch)
7. Setup realtime subscriptions
8. Render globe with all data

#### Realtime Updates (Supabase WebSockets)
- `rightnow-realtime` â†’ Right Now status changes
- `beacons-realtime` â†’ Beacon CRUD operations
- `user-activities-realtime` â†’ Activity tracking
