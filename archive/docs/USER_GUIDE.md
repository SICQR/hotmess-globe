## 1. Getting Started

### Creating Your Account

1. Visit [hotmess.london](https://hotmess.london)

**Important**: HOTMESS is 18+ only. You must be at least 18 years old to use the platform.

### First Steps

After creating your account:

2. **Set up safety features** - Add emergency contacts
3. **Explore events** - Find what's happening near you
4. **Connect with people** - Send handshakes to make connections
5. **Join the community** - Engage with posts and discussions

## 3. Events & Beacons

### Finding Events

**Browse Events:**
1. Go to **Events** page
2. Filter by:
   - Location
   - Date
   - Type (club, concert, party, etc.)
   - Distance from you

**Event Details:**
- When & where
- Description
- Attendee count
- Organizer info
- Check-in code

### RSVP to Events

1. Click on an event
2. Tap **RSVP**
3. Add to your calendar (optional)
4. Set up safety check-in (recommended)
5. Invite friends to join

### Creating Events

**Steps:**
1. Go to **Beacons** â†’ **Create**
2. Choose **Event** type
3. Fill in details:
   - Event name
   - Date & time
   - Location
   - Description
   - Capacity
4. Add cover image (optional)
5. Set visibility
6. Publish

**Tips:**
- Clear, descriptive titles
- Accurate location
- Include safety information
- Update if details change
- Respond to attendee questions

### Checking In

**At the Event:**
1. Open HOTMESS app
2. Go to **Scan**
3. Scan event QR code OR
4. Manual check-in from event page

**Benefits:**
- Earn XP
- See who else is there
- Access event-specific features
- Track your event history

### Beacons (Drops)

Beacons are location-based drops you can create:
- **Events**: Parties, concerts, meetups
- **Hangouts**: Spontaneous gatherings
- **Meetups**: One-on-one or small groups
- **Pickups**: Share rides, meet before events

**Creating Beacons:**
1. Select type
2. Set location and duration
3. Add description
4. Set who can see it
5. Drop it!

## 6. Safety Features

HOTMESS prioritizes your safety. Use these features to stay safe:

### Safety Check-Ins

**What it is:**
A timed reminder to confirm you're safe when meeting someone.

**How to use:**
1. Before meeting someone, enable check-in
2. Set time (e.g., 2 hours)
3. Receive reminder notification
4. Confirm you're safe
5. If you don't respond, emergency contacts are notified

**Setting Up:**
- Go to **Care** page
- Set check-in time
- Add meeting details

### Emergency Contacts

**Add Emergency Contacts:**
1. Go to **Care â†’ Emergency Contacts**
2. Add trusted people
3. They'll be notified if:
   - You use the panic button
   - You miss a safety check-in
   - You request emergency help

### Panic Button

**Location:** Bottom of screen (when enabled)

**What it does:**
- Immediately notifies emergency contacts
- Shares your location
- Calls emergency services (if you choose)

**When to use:**
- You feel unsafe
- Emergency situation
- Need immediate help

**Note:** This is for real emergencies only.

### Blocking & Reporting

**Block a User:**
1. Go to their profile
2. Tap menu (â‹¯)
3. Select **Block**
4. They can't contact or see you

**Report Content:**
1. Tap menu (â‹¯) on content/profile
2. Select **Report**
3. Choose reason:
   - Harassment
   - Inappropriate content
   - Fake profile
   - Spam
   - Safety concern
4. Submit

**What happens:**
- Our moderation team reviews within 24 hours
- Safety issues prioritized
- Action taken if violation confirmed
- You'll be notified of outcome

### Meeting Safely

**Best Practices:**
- Meet in public places
- Tell someone where you're going
- Use safety check-ins
- Trust your instincts
- Don't share too much personal info early
- Keep conversations on platform initially
- Have your own transportation

## 9. Troubleshooting

### Common Issues

**Can't Log In:**
- Check email/password are correct
- Try password reset
- Clear browser cache
- Check if account is locked
- Contact support if persists

**App Performance:**
- Close and reopen app
- Update to latest version
- Clear app cache
- Check internet connection
- Restart device

**Location Not Working:**
- Enable location services in device settings
- Grant location permission to HOTMESS
- Check if GPS is working
- Try restarting device

**Payment Failed:**
- Verify card details
- Check if card has sufficient funds
- Try different payment method
- Contact your bank if blocked
- Contact support@hotmess.london

**Not Receiving Emails:**
- Check spam/junk folder
- Add noreply@hotmess.london to contacts
- Verify email address in settings
- Check notification preferences

### Getting Help

**Help Center:**
- Visit `/help` for FAQs

**Contact Support:**
